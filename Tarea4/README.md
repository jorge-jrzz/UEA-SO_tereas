# Tarea 4

## Install dependency

```bash
pipenv install
pipenv shell
```

## Run the programs

#### Only the frontend

```bash
python front.py
```

#### Using `multiprocessing.Manager` to share data between processes

```bash
python -m manager.main
```

#### Using `multiprocessing.shared_memory` to share data between processes

```bash
python -m shared_memory.main
```
